Bem-vindo(a) ao meu repositório! Aqui estão reunidos todos os algoritmos, exercícios e desafios que desenvolvi ao longo do meu **1º Semestre em Sistemas de Informação**. 

Este repositório tem como objetivo apresentar minha evolução e aprendizado em lógica de programação e no ecossistema Python.

---

## 📂 Organização do Repositório

O projeto está estruturado de forma cronológica por semanas acadêmicas, acompanhando a evolução da complexidade dos conceitos:

| Pasta | Descrição dos Conceitos Aprendidos |
| :--- | :--- |
| `Semana01` ao `Semana3` | Conceitos fundamentais, variáveis, operadores matemáticos e estruturas condicionais (`if/else`). |
| `Semana4, O Inimigo Agora é Outro` ao `Semana9` | Onde o jogo ficou sério! Estruturas de repetição (`while` e `for`) e controle de fluxo complexo. |
| `Semana10` ao `Semana11` | Manipulação de listas e introdução a funções. |
